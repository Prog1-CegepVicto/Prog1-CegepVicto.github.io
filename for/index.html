    Boucle : for

    La boucle for fait une nombre déterminé de répétitions.

    Elle est utilisée quand on sait exactement le nombre d'opérations à effectuer.

    Syntaxe

    for(variable pour compter; condition de fin; modification de la varible pour compter)
{
    // instructions
}

    En C#

    Par convention (et en lien avec les mathématiques) les compteurs utilisés sont habituellement : i, j, k, u, v, w

    Mais il est possible de nommer la variables autrement tant que l'on respecte les règle de nomenclature d'une variable

    // Cette boucle affiche de 0 à 9 à l'écran
for(int i = 0; i < 10; i++)
{
    Console.WriteLine(i);    
}

    Condition de fin

    La boucle arrête quand la condition de fin a une valeur logique : FAUX

    Il n'est pas obligatoire de toujours utiliser une incrémentation de 1

    Répéter des instructions

    For sert souvent à répéter des instructions un nombre de fois déterminé.

    La variable qui sert de compteur joue aussi fréquement un rôle dans les instructions.

    Par exemple cette boucle effectue 8 fois les instructions entre les accolades

    string motDePasse = "";
for(int i = 0; i < 8; i++)
{
    motDePasse = motDePasse + i;
}
    • Ici le mot de passe commence vide.
    • i est la variable qui sert de compteur, elle commence à 0 et augmente de 1 à chaque fois que les instructions de la boucle sont exécutées.
    • Les instructions de la boulces sont exécutées autant de fois que i est plus petit que 8.
    • À chaque fois que les instructions de la boucle sont exécutées on ajoute i au string.
    • À la fin de la boucle le mot de passe contient "01234567".

    Utilisation avec les tableaux

    Dans la grande majorité des cas, la boucle for est utilisée pour parcourir un tableau de la permière à la dernière case.

    La variable compteur (i, j, k, etc..) est utilisée pour désigner la case (l'index) du tableau à traiter.

    Un tableau commence toujours à l'index (numéro de la case) 0, donc on donne au compteur (ex: i, j, k) la valeur de 0 pour commencer.

    Un tableau fini toujours à la case dont l'index est : le nombre d'éléments (cases) du tableau - 1. Donc on met la condition de fin de la boucle à : compteur < nombre d'élements.

    // Boucle générale pour parcourir un tableau
// Ici on assigne un nombre aléatoire de 0 à 100 pour chaque élément du tableau
Random rng = new Random();
int[] tab = new int[30];

for(int i = 0; i < tab.Length; i++)
{
    tab[i] = rng.Next(0,101);
}
