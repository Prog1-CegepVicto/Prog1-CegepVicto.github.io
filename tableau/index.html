    Structure de données : tableau

    Le tableau permet d'avoir plusieurs valeurs stockées en mémoire pour une seule variable.

    Les données stockées dans le tableau doivent être de la même nature comme :

    • Une série de noms
    • Une série de notes
    • Les prix des items dans un panier

    Le tableau permet entre autre :

    • Faciliter le traitement de variables multiples (ex.: note1, note2, note3, ...)
    • Utiliser une boucle pour traiter plusieurs données
    • Améliorer le temps d'accès au données (Les données d'un tableau sont placée une à côté de l'autre en mémoire)

    Déclaration

    type[] nom = new type[nombre d'élément]

    Le nombre de données dans un tableau doit être fixée lors de la déclaration

    Exemple en C#

    int[] notes = new int[30];

    Déclaration alternative

    Quand on sait déjà ce qu'il y a dans le tableau, on peut déclarer le tableau différement.

    int[] notes = new int[]{98, 76, 42, 73, 89}; ou int[] notes = {98, 76, 42, 73, 89};

    Utiliser un tableau

    • Comme les variables, les valeurs des tableaux peuvent être utilisées pour des calculs ou modifiées.
    • Pour accéder à une donnée du tableau, il faut indiquer sa position.
    • La position d'une donnée dans le tableau s'appelle index
    • La numérotation des index du tableau commence toujours à 0
    • On peut obtenir la longueur d'un tableau en utilisant nomDuTableau.Length

    Quelques exemples d'utilisation

    int[] notes = {98, 76, 42, 73, 89};
double moyenne = 0;

// Modifier la première note du tableau
notes[0] = 97;

// Afficher la 3ème notes du tableau
Console.WriteLine(notes[2]);

// Faire la moyenne avec une boucle
for(int i = 0; i < notes.Length; i++)
{
    moyenne = moyenne + notes[i];
}
moyenne = moyenne / notes.Length;

    On utilise souvent une boucle pour accéder à chacune des valeurs du tableau!
