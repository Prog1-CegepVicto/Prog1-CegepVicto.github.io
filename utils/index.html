    Utilitaires et essentiels

    Afficher à la console sans sauter de ligne

    Console.Write("Mon texte.");
Console.Write(" La suite de mon texte.");

    Effacer la console

    Console.Clear();

    Inclure un saut de ligne dans une message de type string

    Le symble \n dans un texte fera sauter une ligne dans un string

    Console.WriteLine("Ma première ligne \n Ma deuxième ligne");

    Saisir à la console

    Texte

    string leTexte = "";

Console.WriteLine("Entrer votre texte");
leTexte = Console.ReadLine();

    Nombre entier (int)

    int nombre = 0;

Console.WriteLine("Entrer votre nombre");
int.TryParse(Console.ReadLine(), out nombre);

    Nombre réel (double)

    double nombre = 0;

Console.WriteLine("Entrer votre nombre");
double.TryParse(Console.ReadLine(), out nombre);

    Générer une nombre aléatoire

    Placer cette initialisation au début de votre méthode

    Random aleatoire = new Random();

    Pour assigner une nombre aléatoire à une variable

    nombre = aleatoire.Next();

    Pour assigner une nombre aléatoire avec un maximum (commence à 0) Le maximum doit toujours être plus grand de 1 que le nombre maximum désiré

    nombre = aleatoire.Next(101);

    Pour assigner une nombre aléatoire avec un minimum et un maximum Le maximum doit toujours être plus grand de 1 que le nombre maximum désiré

    nombre = aleatoire.Next(1, 101);

    Faire une pause

    Thread.Sleep(1000);

    Le paramètre passé à la méthode Sleep est un nombre de millisecondes. Le paramètre ci-dessus fait une pause de 1 seconde.

    Caculer la puissance (exposant) et la racine carré d'un nombre

    Math.Pow(2,5); équivalent de 2 puissance 5, donne 32 Math.Sqrt(4); équivalent de la racine carré de 2, donne 2

    Mettre en majuscules/minuscules une chaîne de caractères (string) : ToUpper(), ToLower()

    string nom = "Denis";
string nomEnMinuscules = "";
string nomEnMajuscules = "";

// Mettre en minuscule

nomEnMinuscules = nom.ToLower();
nomEnMajuscules = nom.ToUpper();

    Lire un fichier de texte (extension .txt)

    On a besoin des outils de traitement des fichiers. Il faut donc ajouter, au début du fichier (avant toutes les autres lignes) using System.IO;

    Un peu comme dans le cas du générateur de nombres aléatoires, il y a une variable à déclarer et initialiser avant de lire le fichier​ StreamReader lecteur = new StreamReader("MonTexte.txt");​

    ATTENTION! :

    • Entre les parenthèses il faut donner le nom du fichier à lire et ce fichier doit être dans le même répertoire que le projet​
    • Dans Visual Studio, dans l'explorateur de solutions, cliquer de droite sur le fichier texte et sélectionner Propriétés
    • À la propriété Copier dans le répertoire de sortie choisir Copier si plus récent

    Enfin, on peut lire une ligne du fichier​ texte = lecteur.ReadLine();​

    Il faut fermer le lecteur quand nos traitements sont terminés, sinon l'ordinateur considère que le fichier est en cours d'utilisation​ lecteur.Close();

    Exemple complet 

    static void Main(string[] args)
{
    string texte = "";
    string cheminFichier = "monTexte.txt";
    StreamReader lecteur = new StreamReader(cheminFichier);

    // Tant que le lecteur n'est pas à la fin du fichier
    while (!lecteur.EndOfStream)
    { 
        texte = lecteur.ReadLine();

        Console.WriteLine(texte);
    }

    lecteur.Close();
}
