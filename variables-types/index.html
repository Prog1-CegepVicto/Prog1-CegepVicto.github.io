    Variables et types

    Variables

    • Les variables associent un espace dans la mémoire avec un nom.
    • Elles sont utilisées pour stocker les données necessaires à la programmation

    Types

    Le type d'une variable définit ce qu'elle contient et l'espace qu'elle utilise dans la mémoire :

    • Un nombre entier : 1
    • Un nombre réel : 2.5
    • Un caractère : a
    • Une phrase (chaîne de caractère) : Reste calme et appelle Fred
    • Une valeur vrai ou faux : true

    Il existe beacuoup d'autres types pour les variables!

    Types de base en C#

    • int : un entier
    • double : un nombre réel
    • char: un caractère
    • string: une phrase
    • bool: vrai ou faux

    Il y a des types plus spécifiques. Il nous permettent d'utiliser plus ou moins de mémoire selon ce que l'on a l'intention de faire avec la variable. Par exemple :

    • int: un entier entre -2147483648 et 2147483647 prend 32 bit
    • short: un entier entre -32768 et 32767 prend 16 bit

    Utiliser une variable

    1. La déclarer : La déclaration dit à notre processeur de lui réserver un espace dans la mémoire vive
    2. Lui assigner une valeur : L'assignation met une valeur dans la mémoire

    Quand on combine la déclaration et l'assignation on initialise la variable.

    Déclaration

    • Quand on déclare la variable il est obligatoire de déterminer son type en premier, puis de lui donner une nom
    • Le nom d'une variable doit être significatif. On sait à quoi elle sert en lisant le nom. Par exemple : moyenne, total, texteIntro, ...
    • Le nom d'une variable commence par une lettre minuscule
    • Si le nom d'une variable a plus qu'un mot, tous les mots après le premier mot commencent par une majuscule

    Quelques exemples de déclaration :

    • int total;
    • double moyenne;
    • string prenom;

    Assignation

    • L'assignation se fait avec l'opérateur =
    • Il faut que la valeur assignée corresponde au type de la vairable

    Quelques exemples d'assignation (avec la déclaration avant) :

    string prenom;
prenom = "Jean"

    double moyenne;
moyenne = 78.23;

    On ne répète pas le type lors de l'assignation

    Initialisation

    L'initialisation combine la déclaration et l'assignation en une seule instruction

    Il est important d'initialiser les variables pour éviter la possibilité d'erreur

    Quand on ne sait pas d'avance quelle valeur assigner, on met une valeur par défaut :

    • int : 0
    • double : 0.0
    • char : ''
    • string : ""
    • bool : true

    Quelques exemple d'initialisation :

    int total = 0;
double moyenne = 0.0;
char code = '';
string prenom = "";
bool valide = true;
