    Programmation orientée objet

    La programmation orientée objet ou POO est une méthode de qui consiste à définir des entitées logicielles et leurs intéractions.

    Ces entités, les objets, rerpésentent soit : - Un concept (ex: Contrôleur, l'objet contrôle le déroulement du programme) - Une abstraction du monde physique ou imaginaire (Un personnage, une voiture, un rendez-vous)

    L'objet a : - Un état : sa structure interne, ses attributs (varibables, tableaux, autres objets) - Des comportements: ses méthodes pour modifier l'état ou interagir avec d'autres objets

    L'objectif est donc de représenter les objets et les relations pour former un système qui répond aux besoin du client.

    La classe

    Pour créer un objet, on définit sa classe (class). La classe est une sorte de plan qui nous permettra de construire l'objet et l'utiliser.

    La classe contient :

    • Les attributs : les caractéristiques de l'objet
    • Les méthodes : les comportements et interactions de l'objet

    Exemple : le personnage de jeu vidéo

    Attributs : vie, arme, armure, abiletés, etc... Méthodes : avancer, reculer, attaquer, etc...

    L'instance

    Pour utiliser notre classe, il faut créer une instance de celle-ci.

    Chaque classe publique d'un projet peut être utilisée comme type. Sans le savoir nous avons déjà utilié une classe pour créer un objet (instance)

    Random rng = new Random();

    Cette instruction crée une instance, nommée rng, de la classe Random. On peut maintenant utiliser les fonctionnalités de la classe Random avec la variable rng.
