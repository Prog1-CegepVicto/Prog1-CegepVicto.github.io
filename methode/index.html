    Méthodes

    Un méthode est une série d'instructions qui effectue une une tâche spécifique.

    Elle peut simplement exécuter les instructions mais elle peut aussi nous envoyer une résultat.

    La méthode avec une valeur qui nous revient est une fonction. Par exemple : Console.Readline() nous renvoie un string

    La méthode sans valeur de retour est une procédure : Console.WriteLine(monTexte) affiche mais nous n'avons pas de valeur à assigner

    La méthode Main

    Lorsque l'on crée un nouveau projet avec Visual Studio, il ajoute automatiquement un méthode Main.

    Cette méthode est exécutée au démarrage du programme. Elle contient toutes les instructions et les méthodes personnalisées du programmeur.

    using System;

namespace ExercicesSemaine1
{
    internal class Program
    {
        static void Main(string[] args)
        {

        }
    }
}

    Déclaration

    Une méthode est déclarée à l'intérieur d'une classe.

    Une méthode est à l'extérieur des autres méthodes.

    Dans le programme principal, il faut toujours ajouter le mot clé static avant le type de la méthode.

    Une méthode commence par le type de retour, suivi de son nom et de parenthèses.

    Quand une méthode n'a pas de retour (de résultat) son type est void.

    Une méthode a ses propres variables, elle ne peut pas utiliser les variables d'une autre méthode directement.

    Une méthode est une action, son nom est un verbe et commence par une minuscule

    Procédure : méthode qui effectue un travail mais ne retourne pas de résultat

    using System;

namespace ExercicesSemaine1
{
    internal class Program
    {
        static void Main(string[] args)
        {

        }

        // nouvelles méthodes 

        // méthode qui effectue une tâche mais ne donne pas de résultat
        // static est avant le type puisqu'il s'agit du programme principal
        // la méthode ne retourne pas de résultat donc son type est void

        static void afficherMessage()
        {
            string message = "Bonjour";
            Console.WriteLine(message);
        }
    }
}

    Fonction : méthode qui retourne une résultat

    L'instruction pour renvoyer un résultat est : return.

    Pour utiliser le résultat d'une fonction on peut:

    • Assigner le résultat à une variable
    • Utiliser la fonction à la place d'une variable dans une instruction

    Quand une méthode retourne un résultat il faut désigner le type de retour dans sa déclaration

    using System;

namespace ExercicesSemaine1
{
    internal class Program
    {
        static void Main(string[] args)
        {
            double moyenne = 0;

            // Assignation du résultat à une variable
            moyenne = calculerMoyenne();

            // Utilisation directe dans une instruction
            Console.WriteLine(calculerMoyenne());
        }

        // nouvelles méthodes 

        // méthode qui retourne un résultat
        // static est avant le type puisqu'il s'agit du programme principal
        // la méthode retourne un résultat donc son type est le même que le type de la valeur retournée

        static double calculerMoyenne()
        {
            double note1 = 66;
            double note2 = 95;
            double note3 = 78;
            double moyenne = 0;

            moyenne = (note1 + note2 + note3) / 3;

            return moyenne;
        }
    }
}

    Paramètres de fonctions

    Les méthodes sont toujours suivient de parenthèses.

    Elles servent à transmettre de l'information pour qu'elle puisse faire son travail.

    Par exemple, le contenu des prenthèses dans Console.WriteLine("Mon message") dit à la méthode quel message afficher.

    Ce qui est envoyé à la méthode doit être défini lors de la déclaration de la méthode.

    Par exemple si une méthode a besoin d'un nombre entier pour faire son travail la déclaration serait :

    static void maMethode(int nombreEntier)
{
    // instructions
}

    Quand on utilise la méthode dans notre Main, on sera obligé de donner un nombre entier (ou une variable contenant un nombre enteir) à la méthode :

    // Exemple 1 
maMethode(7);

// Exemple 2 avec une variable
int variable = 5;
maMethode(variable);

    La valeur donnée dans le main est copiée dans la variable paramètre de la méthode.

    Déclaration d'une méthode avec des paramètres

    static void calculerExposant(int nombre, int exposant)
{
    int resultat = 0;

    resultat = Math.Pow(nombre, exposant);

    Console.WriteLine("Résultat : " + resultat);
}

    Ici, les paramètres sont nombre et exposant, de type int

    Chacune des variables déclarée de cette façon sera utilisable uniquement dans la méthode.

    Utilisation d'une méthode avec paramètres

    On utilise une méthode avec paramètres de la même façon que les autre.

    Les paramètres sont séparés par des virgules.

    Il faut lui donner les paramètres dans l'ordre où ils ont été déclarés.

    Si on utilise la méthode précédente :

    calculerExposant(2, 5) Le résultat à la console sera 32

    calculerExposant(5, 2) Le résultat à la console sera 25
